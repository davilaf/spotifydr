# App de sugerencias de canciones con IA para Spotify

Esta app fue creada en Google AI Studio. Su objetivo es:
- Analizar una playlist de Spotify.
- Sugerir canciones que encajen con el estilo de la playlist.
- Ayudar al usuario a descubrir nueva música.

## Cómo funciona la IA

- La IA recibe información sobre la playlist.
- Genera sugerencias de canciones basadas en gustos y estilo.
- El usuario puede aceptar o ignorar las sugerencias.

> Nota: Las recomendaciones son generadas automáticamente por la IA.  
> Si las canciones que ya tienes en tu playlist te gustan, está perfecto: puedes seguir disfrutándolas tal como están.

Para ver el prompt completo y las reglas de la app, consulta el archivo `prompt.md` en este repositorio.
